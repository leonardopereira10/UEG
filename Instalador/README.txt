# Instruções de instalação ------
    1) Abra o terminal (Ctrl+Alt_T)
    2) Navegue até a pasta onde está localizado o arquivo "install.sh"
    3) Marque-o como executável com o comando: chmod +x install.sh
    4) Rode o script install.sh com o comando: ./install.sh


# Instruções desinstalação ------
    1) No terminal, execute o script deinstall.sh
        Não se esqueça de marca-lo como executável (chmod +x deinstall.sh)
